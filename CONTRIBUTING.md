## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes and add or update tests
4. Run the tests: `npm test` or `npm run test:coverage`
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

1. Install [Node.js (18+ recommended)](https://nodejs.org/)
2. Clone the repository: `git clone https://github.com/bellmano/umami-github-badges.git`
3. Install dependencies: `npm install`
4. Run the development server (if applicable): `npm run dev` or `npm start`

## Testing

- Add or update unit tests for new features or bug fixes
- Ensure all tests pass before submitting a PR (`npm test` or `npm run test:coverage`)

## Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/bellmano/umami-github-badges/issues).
